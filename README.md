## Getting Started

### Installing
#### Ubuntu
Clone the repo and run:
```
$ make
```

Try it out by running:
```
$ echo lineparine | ./akrantiain2 samples/sample_lineparine.snoj
```

#### Windows
1. Install Haskell Platform (>= 7.10.2).
2. Clone the repo.
3. Run:
```
$ ghc --make akrantiain2.hs
```

Try it out by running:
```
$ echo lineparine | akrantiain2 samples\sample_lineparine.snoj
```
